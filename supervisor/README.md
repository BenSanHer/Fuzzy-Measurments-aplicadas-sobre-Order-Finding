# Observaciones del supervisor

En esta sección los profesores supervisores de este proyecto pueden colocar sus observaciones generales y sus anotaciones a manera de notas directamente. 

## Metodología

Para dar alguna anotación u observación, genera un pull request con un archivo .md que tenga de titulo Observaciones-dd-mm-aa-apellido para identificar cual es la observación pendiente con la que quiera que trabajemos
